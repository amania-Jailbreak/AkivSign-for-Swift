# AkivSign for Swift

[Web 版 AkivSign](https://sign.amania.jp/) のネイティブ iOS アプリ版

---

## 概要

**AkivSign for Swift**は、[Web 版 AkivSign](https://sign.amania.jp/)のネイティブ iOS アプリケーション版です。

Feather のようなアプリですが、大きな違いは**基本的な処理をサーバーサイドで実行**する点です。  
IPA、dylib、および署名処理はすべてサーバー上で管理されます。

> **注意**: かなり手抜き工事のバックエンドのため、個人情報などの重要な情報はアップロードしないでください。

---

## 主な機能

### アプリ管理

-   サーバーにアップロードされた IPA ファイルの一覧表示
-   アプリ情報の詳細表示
-   署名済み IPA のインストール

### 証明書管理

-   開発者証明書（`.p12`）のアップロード
-   プロビジョニングプロファイル（`.mobileprovision`）のアップロード
-   証明書一覧の表示と選択
-   証明書の追加・削除
-   プロビジョニング情報の詳細表示
    -   証明書名
    -   UUID
    -   App ID
    -   Team ID
    -   有効期限

### Tweak 管理

-   利用可能な dylib の一覧表示
-   複数 dylib の選択と注入
-   dylib のアップロード

### リポジトリ検索

-   AltStore向けレポジトリの検索 (なお現在はユーザー自身でリポジトリを管理することはできませんそのうちやります)

> 検索タブからのインストールは**サーバー内で全て完結**するため、低速回線でも利用可能です（インストール自体は別）

### 共有シートからのアップロード

-   他のアプリから IPA ファイルを受け取り
-   共有メニューからの直接アップロード

## 使い方

### 初回セットアップ

1. **ログイン**  
   アプリを起動してログインします

    > できれば Discord ログインを使用してください（他の方法は多分バグります Discordの方が安全だしね）

2. **証明書の追加**  
   プロフィールタブで新しい証明書を追加

3. **証明書ファイルのアップロード**
    - `.p12`ファイルをアップロード
    - プロビジョニングプロファイル（`.mobileprovision`）をアップロード
    - `.p12`のパスワードを設定

### IPA ファイルのインストール

#### ステップ 1: IPA のアップロード

以下のいずれかの方法で IPA ファイルをアップロード/ダウンロードします：

-   **ホームタブ**: IPA ファイルを直接アップロード
-   **検索タブ**: レポジトリから検索してダウンロード
-   **共有シート**: Safari 等から IPA ファイルを共有

#### ステップ 2: 署名とインストール

1. ホームタブから追加した IPA ファイルの「Install」ボタンを押す
2. 使用する証明書を選択
3. 必要に応じてオプションを調整：
    - **Tweak 注入**: 必要な dylib を選択
    - **File Picker Fix**: ファイルピッカーの不具合修正が必要な場合
    - **PPQ Check Bypass**: 推奨オプション
4. サーバー上で署名が実行されます
5. 完了後、「インストール」ボタンでインストール

## サポート・注意事項

### 動作要件

-   **OS**: iOS 26.0 以降
-   **接続**: インターネット接続必須
-   **証明書**: 有効な Apple Developer Program 証明書

### 重要な注意事項

> **セキュリティ**
>
> -   開発者証明書は自己責任で管理してください
> -   ユーザートークンは他人に共有しないでください
> -   個人情報や機密情報はアップロードしないでください

> **制限事項**
>
> -   署名できるアプリは証明書のプロビジョニングプロファイルに依存します
> -   このサービスを利用したことによる損害は責任を負いかねます
> -   基本的に手抜き実装のため、バグが存在する可能性があります

> **バグ報告**
>
> -   バグを発見した場合は、Issue でお知らせください
